Roku Web Remote

Provide the IP Address of your Roku Device and control it using the interface.

The Roku Device must be connected to the same network of the device this page is being viewed on.

Just place the IP Address in the text field and select a button below.

  • From the Roku main menu, move down to the settings.
  • Search for the networking option.
  • Under that submenu, find About. This section will include the Roku Device's IP Address.
